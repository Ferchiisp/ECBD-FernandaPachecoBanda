## Descripcion:
Repositorio académico que concentra las prácticas y laboratorios desarrollados en la asignatura Extracción de Conocimiento de Bases de Datos. Incluye actividades de limpieza de datos, análisis exploratorio, visualización de información, interpretación de resultados y generación de conocimiento a partir de distintos conjuntos de datos públicos.

## Objetivo:
Integrar en un repositorio organizado las evidencias de aprendizaje de la asignatura mediante el uso de técnicas de preparación, análisis y visualización de datos, aplicando buenas prácticas de documentación, control de versiones y generación de conocimiento basadas en el modelo DIKW (Datos, Información, Conocimiento y Sabiduría).

## Instrucciones de ejecución general:
### 1. Clonar el repositorio
Abrir una terminal y ejecutar el siguiente comando:
[git clone https://github.com/Ferchiisp/ECBD-FernandaPachecoBanda.git

Posteriormente acceder al directorio del proyecto:
cd ECBD-FernandaPachecoBanda

### 2. Instalar Anaconda
Verificar que Anaconda se encuentre instalado en el equipo. En caso de no contar con la herramienta, descargarla e instalarla desde el sitio oficial.

### 3. Abrir Anaconda Navigator
Ejecutar Anaconda Navigator y seleccionar el environment Extraccion-9BESC la herramienta Jupyter Notebook.

### 4. Acceder al repositorio
Desde Jupyter Notebook navegar hasta la carpeta del repositorio clonado.

### 5. Verificar los datasets
Comprobar que los archivos de datos se encuentren dentro de la carpeta DataSet:
dataset_sucio_practica.csv
netflix_titles.csv
test.csv
ventas-por-factura.csv

### 6. Ejecutar los notebooks
Ingresar a la carpeta Notebooks y abrir el laboratorio correspondiente:
 -- Lab01.ipynb
 -- Lab02.ipynb
 -- Lab03.ipynb
...

Ejecutar las celdas en orden utilizando la opción:
Kernel → Restart & Run All
o bien ejecutarlas manualmente de forma secuencial.

### 7. Visualizar resultados
Al finalizar la ejecución se mostrarán:
 -- Procesos de limpieza de datos.
 -- Estadísticas descriptivas.
 -- Gráficas y visualizaciones.
 -- Interpretaciones de resultados.
 -- Conclusiones derivadas del análisis.
